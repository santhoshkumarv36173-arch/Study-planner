# Study Planner

Simple Study Planner web application built using Flask.

## Features

- User Login & Register
- Create Study Plans
- Add Tasks
- Track Progress
- Simple Dashboard

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Project Structure

```bash
Studyplanner/
│
├── static/
├── templates/
├── main.py
├── models.py
├── requirements.txt
└── README.md
```
## Installation

Clone the repository: git clone https://github.com/santhoshkumarv36173-arch/Study-planner.git

#### Go to project folder:   cd Study-planner

#### Create virtual environment:  python -m venv venv

#### Activate environment:  venv\Scripts\activate

#### Install packages:  pip install -r requirements.txt

#### Run project:  python main.py

## Open in Browser
### http://127.0.0.1:5000
